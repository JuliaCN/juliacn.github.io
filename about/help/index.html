New to Julia?

  • Read the Getting Started tutorial.

  • Do you learn best via programming books? Check out the full list of Julia Books.

Looking for code?

  • See the downloads page for links to the Julia Binaries.

  • Explore the development repository on GitHub.

Have a Julia question?

  • First check the Julia FAQs section of the docs which includes answers to many common, general Julia questions.

  • The next best place to go to ask a question is the offical Julia Discourse site.

  • Stack Overflow has a growing number of Julia questions and answers.

  • For realtime help from a person, you should check out our official chat channels.

  • If you think there may be a bug in core Julia, search the open issues on the core Julia repo.

  • If you think you may have found a security vulnerability in Julia, please read the instructions for reporting security issues.

  • If you think you have found a problem with this website, please see the issues section for the website repository.

Looking for a particular Julia Package?

  • Check out the packages page to browse the complete list of all registered Julia Packages.

Want to contribute?

  • To report a bug in core Julia, use the Julia issue tracker on GitHub.

  • To report a problem with this website, see the issues section for the website repository.

  • To contribute a bug fix or other patch to core Julia, see the Julia Developer's Guide.

  • To contribute to the official Julia documentation, use the contributing guide to contribute a documentation patch.

  • To contribute to the official Julia Language website, see the README for the website repository.

  • To announce your package to the Julia community, post on the Julia Discourse under the package announcement section.

Need to contact the Julia Language?

  • Contact contact@julialang.org and let us know how we can help!

Anything else?

  • Contact logan@julialang.org and let us know how we can help!
