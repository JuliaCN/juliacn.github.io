Julia Packages

The Julia ecosystem contains over 3000 packages which means finding the right package can be a challenge. Fortunately, there are services that can help navigate the ecosystem, including:

  • JuliaHub — a Julia Computing service that includes search of all registered open source package documentation, code search, and navigation by tags/keywords. It is powered by Julia Team.

  • JuliaObserver — see what packages are popular and/or trending, navigate by package categories.

  • Julia.jl — a manually curated taxonomy of Julia packages (category information for JuliaPackages is derived from this as well).
